The **R&R Dashboard** is an analytics platform designed to visualize and analyze employee recognition data from the Kudos Bot system. It leverages VADER sentiment analysis to transform raw feedback into actionable insights, helping organizations build positive workplace culture through data-driven recognition patterns.

### ğŸ¯ Key Highlights

- **VADER Sentiment Analysis**: Automatically analyzes all comments with compound scores converted from -1 to +1 range into an intuitive 0-100 index
- **Real-time Analytics**: Track recognition trends, engagement metrics, and sentiment distribution across teams
- **Interactive Dashboards**: Built with Streamlit for seamless user experience
- **Comprehensive Insights**: Identify top contributors, recognition patterns, and areas for improvement
